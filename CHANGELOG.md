## [1.0.0] - 2026-01-12

### Added

- Initial release of photos-mcp
- MCP server implementation for Apple Photos on macOS
- Browse albums functionality to list all photo albums
- Search photos capability with various filters
- Export images to specified locations
- Full Model Context Protocol (MCP) compliance
- TypeScript implementation with strict type checking
- Comprehensive test suite with Vitest
- GitHub Actions CI/CD pipeline
- ESLint and Prettier for code quality
- Husky pre-commit hooks for automated linting
